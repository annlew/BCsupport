 VSCode on Tetralith
=====================

Notebook through VSCode
+++++++++++++++++++++++

Start VSCode in your local machine. You will need the following extensions:

* Jupyter
* Remote - SSH
* Remote  Explorer
* Python

Connect to Tetralith by clicking the button in the lwer left corner "Open a Remote Window".
